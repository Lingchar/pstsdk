Win32

  1. Install Visual Studio 2010
  2. Install SVN to C:\svn, learn how to use it
  3. Install MinGW Make to C:\mingw. I renamed C:\mingw\bin\mingw32-make.exe to just make.exe for sanity's sake
  4. Check out the current trunk to C:\libraries\fairport
  5. Install (unzip) Boost to C:\libraries\boost
  6. Create a batch file which:
    1. Executes the "Visual Studio 2010 Command Prompt" batch file
    2. Adds C:\svn to %PATH%
    3. Adds C:\mingw\bin to %PATH%
    4. Adds C:\libraries\fairport and C:\libraries\boost to %INCLUDE%
    5. Drops you into C:\libraries\fairport
  7. Try getting the "test suite" to compile (C:\libraries\fairport\test) by running make

Once you're setup and building, you should take a look at the Big Picture Issues to see areas where you can help out.
